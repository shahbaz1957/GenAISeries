# Node.js Gemini Chat + Streaming Response Example

This project demonstrates how to use **Google's Gemini (via `@google/genai`)** in Node.js to:

- Create a persistent chat session
- Send user input interactively
- Generate **streaming AI responses** in real-time

## Features

- Interactive CLI prompt for asking questions
- Chat session with history support
- Streaming text output (token-by-token response)
- Environment variable-based API key handling

## Requirements

- Node.js 18+
- A Gemini API Key from Google AI Studio: <https://aistudio.google.com/>
- `npm`
